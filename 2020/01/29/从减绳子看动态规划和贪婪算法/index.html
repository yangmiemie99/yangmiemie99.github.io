从减绳子看动态规划和贪婪算法

发表于 2020-01-29

剑指Offer14 减绳子

给你一根长度为n的绳子，请把绳子剪成整数长的m段（m、n都是整数，n>1并且m>1），每段绳子的长度记为k[0],k[1],…,k[m]。请问k[0]xk[1]x…xk[m]可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

动态规划解法

问题分析

  • 这个问题的目标就是求出各段绳子长度乘积的最大值，也就是问题的一个最优解——这是可以应用动态规划求解的第一个特点。
  • 我们把长度为n的绳子剪成若干段的乘积最大值定义为f(n)。假设第一刀剪在长度为i(0<i<n)的位置，于是把绳子剪成了i和n-i两段。我们要想得到整个问题的最优解那么我们就需要用同样的优化方法使得长度为i 和n-i的两段分别剪成若干段。使得各自剪出的每段绳子的长度积最大。也就是争议问题的最优解依赖各个子问题的最优解，这是可以应用动态规划求解问题的第二个特点。
  • 我们把大问题分解为小问题，这些小问题还有相互重叠的跟小的问题，这是动态规划求解问题的第三个特点。
  • 子问题在分解的大问题中重复出现，可以从下向上的将小问题的最优解存储起来，从上向下分析问题，从下向上求解问题，这是求解动态规划化问题的第四个特点。

该问题经过分析递推关系式可以写为$f(n) = \max_{0<i<n} {(f(i)*f(n-i))}$

贪婪算法

问题分析

经过分析，我们可以按照如下的策略切割绳子：当 n >= 5时，我们尽可能的减长度为3的绳子;当绳子长度为4时，把绳子减成两段2。

证明：当 n >= 5的时候，我们证明 2(n-2) > n ，并且 3(n-3) > n。也就是说，当绳子剩下的长度大于5的时候，我们就把他剪成长度为3或者2的绳子。另外当n >= 5的时候 3(n-3) >= 2(n-2)，因此我们减长度为3的绳子。当绳子长度为4的时候，如果要减一刀我们要剪为 2 ×2 ，如果绳子长度大于4我们也可以没必要剪。
